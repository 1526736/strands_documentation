  • Use avconv to losslessly compress depth video

Libav version 9 features lossless compression of 16 bit greyscale video. To compress images in a folder using the command line, use the command(assuming you've installed libav in your home folder).

(HOME_FOLDER)/libav/bin/avconv -r 30 -i (IMAGE_FOLDER)/depth%06d.tiff -pix_fmt gray16 -vsync 1 -vcodec ffv1 -coder 1 video.mov

(HOME_FOLDER)/libav is the location of your libav installation. -r is the option for framerate, in this case 30. The codec used is the lossless ffv1 codec. To use 16 bit greyscale video we need the option -coder 1. The images can be in either png or tiff 16 bit format (tiff doesn't always work for some reason). You have to supply the path to your images in the format specified above.

To decode the video, use the following command.

(HOME_FOLDER)/libav/bin/avconv -i video.mov -f image2 -pix_fmt gray16 (RESULTS_FOLDER)/depth%06d.tiff

Here you need to supply the formatting to the resulting images in the same way as above.
